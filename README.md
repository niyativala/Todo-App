# 📝 ToDo App

A simple and responsive ToDo App built using HTML, CSS, and JavaScript. 
It allows users to add, delete, and manage daily tasks efficiently.

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 💻 How to Run
1. Clone the repository
2. Open index.html in your browser
